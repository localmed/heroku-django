# Heroku Django

Because...

``` bash
$ heroku run python manage.py syncdb
```

...takes way too long to type. How about:

``` bash
$ heroku django syncdb
```

## Installation

``` bash
$ heroku plugins:install https://github.com/localmed/heroku-django.git
```
